
MindM8 - Netlify Deployment with Serverless GPT-4 Function

What's Included:
- index.html
- style.css
- script.js
- netlify/functions/mindm8.js
- README.txt

Deployment Steps:
1. Unzip this archive on your computer.
2. Upload all contents (not the folder) to a GitHub repository.
3. Go to Netlify > "Add new site" > "Import from Git".
4. Connect your GitHub repo and deploy.
